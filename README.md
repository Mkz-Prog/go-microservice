# go-microservice

Тестовый микросервис. Работа с POST и GET запросами.

## Структура проекта

Данный проект представляет из себя простейший веб-сервер, который может обрабатывать POST и GET запросы, а именно:

- проверка работоспособности сервера,
- добавление пользователей,
- просмотр пользователей,
- вывод информации о времени.

## Запуск

```bash
go run main.go
```

После вывода сообщения "server start listening on port 8080..." можно отправлять запросы на сервер. Например:

```bash
curl http://localhost:8080/health
```
